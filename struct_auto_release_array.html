Detailed Description

template<typename T>
struct AutoReleaseArray< T >

Mutable Array type that auto-releases the contained objects upon destruction. T must be necessarily be a pointer.

Template Parameters
TThe type of object to store
Warning
Do not use this type as a return type as the object will be destroyed on function exit.
